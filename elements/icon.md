# Icon

固定會使用到的 icon 已製作成字型檔，使用時帶入 Icon 的樣式名稱即可。在建立新 icon 時，可注意是否滿足以下條件：

1. 重複性高
2. 在 EBU、CBU、E-Service 等服務中都可使用
3. 不需個別微調
